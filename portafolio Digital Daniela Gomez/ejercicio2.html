Ejemplos De Atributos Globales

Los atributos comunes de todos los elementos HTML se puede utilizar en tos los elementos aunque pueden tener efecto enalgunos de ellos los atributos globales pueden especificarse en todos los elementos HTMLe incluso en aquellos no especificados en el estandar Esto significa que cualquier elemento no estandar tambien debe permitir estos atributos aun cuando el uso de tales elementos significa que el documento ya no cumple con HTML5

acceskey

Proporciona y genera un acceso para el documento actual

class

Especifica uno o mas nombres de clase de un elemento ejemplos los subtitulos color red y font-size 15px

dir

Especifica la direccion del texto para el contenido de un elemento observe que el subtitulo dir esta a la derecha

hidden

Especifica que no es todavia o ya no es relevante es decir lo oculta observe que el subtitulo hidden esta oculto

id

Especifica un identificador unico para un elemento ejemplo ver titulo con color blue y font-size 50px

lang

Especifica el lenguaje del contenido del elemento ejemplo este parrafo en espanol

style

Especifica una linea de estilo css de un elemento con style color verde

tabindex

Especifica el orden de la tabulacion de un elemento. El orden comienza en uno.

tabindex

Especifica imformacion adicional acerca de un elemento es editable o no por ejemplo

title

Especifica imformacion adicional acerca de un elemento. la imformacion generalmente sera mostrada por un globo al poner el puntero encima, por ejemplo al ubicarse sobre este subtitulo aparecera el mensaje ¨ejemplo d titulo¨

contenteditable

Espesifica si el contenido de un elemento es editable o no, por ejemplo al dar clic sobre este texto se podra editar

dragable

Especifica si un elemento se puede arrastrar ono, este parrafo se puede arrastrar

spellcheck

Especifica si el elemento ha de tener su ortografia y gramatica marcada o no. los elementos que pueden tener revicion ortografico so:iputs de tipo texto,textareasy elementos en modo editables, por ejemplo al dar doble clic sobre este parrafo se puede ver marcas de ortografia
